Instructions

The calculator can be used by both keyboard and mouse input. For keyboard input, simply start typing and the calculator input will be focused. Use "escape" to clear and "enter" to compute the result.

The functionality of the calculator should feel familiar to any Windows users, at it is designed to mimic the built in Windows calculator. Inputting any operation after a number will cause that operation to be pending. Once another number has been input, the result can be computed either by pressing "enter" or another operation. This way many operations can be chained in a row.

Like in Windows calculator, scientific notation is employed for larger numbers. Editing such numbers is a little funky, as the "e+" or "e-" portion cannot be directly edited, however the preceding float can be changed by clicking on it first to focus it.
